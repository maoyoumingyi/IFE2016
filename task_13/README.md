# 任务十三：零基础JavaScript编码（一）

## 任务描述

> 参考示例代码，补充其中的JavaScript功能，完成一个JavaScript代码的编写
  本任务完成的功能为：用户可以在输入框中输入任何内容，点击“确认填写”按钮后，用户输入的内容会显示在“您输入的值是”文字的右边

## 注意事项

* 实现简单功能的同时，请仔细学习JavaScript基本语法、事件、DOM相关的知识
* 请注意代码风格的整齐、优雅
* 代码中含有必要的注释
* 可以不考虑输入的合法性
* 建议不使用任何第三方库、框架
* 示例代码仅为示例，可以直接使用，也可以完全自己重写
